About Us

This web application is an interface for a PostgreSQL database for businesses. The PostgreSQL database contains more than 100.000 records of demonstration businesses(names, industries, countries, websites ...). Apache Tomcat is the server used for this application, and the server side processing is don by Java 17, which is used to query the database and to process data according to the user's request.

On the client side i have used a combination of HTML for the structure of the web pages, CSS for styling them, and JavaScript for validating the user's input and minor restailing.

Have a nice day and enjoy this application.
